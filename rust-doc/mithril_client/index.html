Define everything necessary to list, download, and validate snapshots from a Mithril Aggregator.

To query an aggregator for snapshots & certificate use the AggregatorHTTPClient that implement the AggregatorHandler trait.

To list, download, and validate snapshots use the Runtime. You must initialize it by giving it a CertificateVerifier, a ProtocolGenesisVerifier and a AggregatorHandler, and a Digester implementations using the with_xxx methods.
