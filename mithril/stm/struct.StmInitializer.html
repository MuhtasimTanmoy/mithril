Expand description

Initializer for StmSigner. This is the data that is used during the key registration procedure. One the latter is finished, this instance is consumed into an StmSigner.

Build the avk for the given list of parties.

Note that if this StmInitializer was modified between the last call to register, then the resulting StmSigner may not be able to produce valid signatures.

Returns an StmSigner specialized to (1) this StmSigner’s ID and current stake (2) this StmSigner’s parameter valuation (3) the avk as built from the current registered parties (according to the registration service) (4) the current total stake (according to the registration service)
